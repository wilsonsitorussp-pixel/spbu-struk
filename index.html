☁️ Sinkronisasi Pengaturan (GitHub)
Konfigurasi struk & riwayat transaksi bisa disimpan ke GitHub repo kamu, sehingga saat dibuka dari HP/perangkat lain otomatis sama dan menyatu. Token disimpan hanya di perangkat ini (localStorage), tidak dikirim ke siapapun selain GitHub API.
🔗 Link Setup Perangkat Lain
Buat 1 link berisi konfigurasi GitHub di atas. Buka link itu sekali di HP/perangkat lain → otomatis tersambung tanpa perlu ketik username/repo/token lagi.

"💾 Simpan Konfigurasi" otomatis upload ke GitHub setiap ada perubahan. Tombol di bawah ini untuk menyamakan semua data antar perangkat (riwayat, profil, operator, logo, harga BBM): menarik dari GitHub, menggabungkan dengan data perangkat ini, lalu mengupload hasil gabungannya kembali.
